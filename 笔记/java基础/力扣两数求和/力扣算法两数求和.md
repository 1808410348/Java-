### 给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标

```
 * 给定 nums = [2, 7, 11, 15], target = 9
 * 因为 nums[0] + nums[1] = 2 + 7 = 9
 * 所以返回 [0, 1]
```



```java
暴力法解法:
    通过双重循环遍历数组中所有元素的两两组合，当出现符合的两数时返回两个元素下标
    /**
     * 遍历----> 暴力破解。。。
     *
     * @param nums      集合数组
     * @param target    目标值
     * @return
     */
        public static int[] findIndexByLoop (int nums[], int target){
        for (int i = 0; i < nums.length; i++) {
            for (int j = i + 1; j < nums.length; j++) {
                if (target - nums[i] == nums[j]){
                    return new int[]{i, j};
                }
            }
        }
        return null;
    }
```



```java
哈希（更优解法）
/**
 * Hash破解
 * @param nums
 * @param target
 * @return
 */
public static int[] findIndexByMap(int nums[], int target){
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int temp = target - nums[i];
        if (map.containsKey(temp)){
            return new int[]{i, map.get(temp)};
        }else {
            map.put(i, nums[i]);
        }
    }
    return null;
}
```
